# 星辰开源机器人
## 架构分析
master-server架构，可实现单机部署

使用到的软件为Python，MySQL，go-cqhttp

## 本机器人目前属于开源阶段
数据处理基本使用Python

功能全部由MySQL数据库存储，由此实现了下属功能的云同步，使用Python的exec语句实现
